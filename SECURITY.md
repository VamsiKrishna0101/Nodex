# Security

Nodex is an agent framework layer. Please report security issues responsibly.

## Reporting a vulnerability

Do not open a public GitHub issue for a vulnerability.

Email the maintainer at:

```text
vklvl0101@gmail.com
```

Include:

- affected version or commit,
- reproduction steps,
- impact,
- suggested fix if known,
- and whether the issue is already public.

## Scope

Security issues may include:

- unsafe execution behavior,
- sensitive data leakage in tracing or display output,
- CLI path handling problems,
- dependency or packaging risks,
- or middleware behavior that bypasses expected checks.

## Supported versions

Until the project reaches a stable release, security fixes target the latest
version on the main branch.
